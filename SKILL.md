# RMRB-Canary — 人民日报政治经济情报框架

## 核心认知（每次分析必读，决定后续所有解读基准）

### 认知一：人民日报是放大器，不是信号源

人民日报的功能是把党中央已定的决策"翻译"给社会，创造合法性，管理预期——它不制造政策，它为政策背书。

真正的政策信号来自：中央文件 → 政治局常委会议 → 领导人批示 → 部委联席决议。人民日报是这些信号的放大和社会化工具。

这个认知有两个关键含义：
- "人民日报沉默"不等于"中央不关注"，可能是在内参渠道、高层窗口会议关注，尚未进入公开传播阶段——这往往是更危险的时刻。
- 我们读人民日报，读到的是"官方决定要社会知道什么"，而不是"官方在想什么"。**两者之间的差值才是最有价值的情报。**

### 认知二：中国政策传导是运动式的，不是线性的

西方政策传导是渐进线性的。中国政策传导遵循：**酝酿期 → 铺垫期 → 动员期 → 运动期 → 收尾期 → 常态化** 六阶段非线性周期。

对企业最有价值的信号在酝酿期和铺垫期，但当前大多数分析工具只能识别运动期（媒体批判密度高峰）。本框架的核心目标是将预警时间前移到铺垫期。

### 认知三：2020年后，"安全"叙事重构了所有行业的政策逻辑

2020年是治理范式的结构性转折点。国家安全叙事从军事政治扩展到经济、数据、粮食、能源、供应链、算法等全域。**当一个行业被纳入安全叙事框架，其政策逻辑从"市场监管"变为"国家治理"——强制力提升，协商余地接近于零，传导速度比普通监管快2-3倍。**

识别目标议题属于哪个叙事框架，比任何打分系统都更决定性。这是本框架新增的"第零步：叙事框架定位"的核心价值。

### 认知四：跨部委协同度是"确定性行动"的最强预测变量

2020年后，网信办+市场监管+公安+财政+税务的联合行动成为标准配置。单部委关注 = 行业风险信号；三部委以上同时发声 = 行动进入倒计时。部委协同度比媒体情感分析更可靠，是识别"酝酿期"和"确定性行动"的关键分界线。

## 分析主流程

### ▶ 第零步：叙事框架定位（新增，所有打分的解读基准）

**在做任何评分之前，先判断目标议题被嵌套在哪个叙事框架里。框架不同，相同信号的含义可以完全相反。**

| 叙事框架 | 识别词汇（高频触发词） | 政策逻辑 | 协商余地 | 传导速度 |
|---------|---------------------|---------|---------|---------|
| **国家安全框架** | 安全、自主可控、卡脖子、供应链安全、数据安全 | 国家治理，非市场逻辑 | 极低 | 极快（压缩50%） |
| **共同富裕框架** | 共同富裕、资本无序扩张、垄断、平台、过度逐利 | 再分配逻辑，针对头部资本 | 低 | 快 |
| **高质量发展框架** | 高质量、绿色低碳、转型升级、新质生产力 | 结构优化，有进有出 | 中 | 正常 |
| **自立自强框架** | 创新、核心技术、国产替代、弯道超车、自立自强 | 支持+规范双重逻辑 | 中高 | 正常 |
| **防范金融风险框架** | 系统性风险、杠杆、债务、流动性、房住不炒 | 金融稳定，防止传染 | 低 | 快 |
| **社会治理框架** | 基层治理、社会稳定、民生保障、群众利益 | 维稳逻辑 | 低-中 | 情景依赖 |

**关键判断**：同一议题可能同时属于多个框架（如新能源 = 自立自强框架 + 防范金融风险框架中的产能过剩子议题）。当叙事框架发生变化（从"自立自强"漂移到"防范风险"），即使其他信号没有变化，风险等级也应当提升。

**输出格式：**
```
叙事框架：[主框架] / [次框架（如有）]
框架变化：[本期与上期相比是否发生框架漂移]
协商余地基准：[极低/低/中/中高]
传导速度调整：[+50%压缩 / 正常 / +30%延长]
```

### 第一层：政策·议程层（看"讲什么"）

**Step 1 — 议程优先度评分**

直接从 `summary.json` 读取 `articles[].agenda_score`。评分规则供核验：

- **版面位置**（0-4 分）：头版头条 4 分，头版非头条 3 分，要闻版 2 分，专题版 1 分
- **篇幅密度**（0-3 分）：超 3000 字 3 分，1500-3000 字 2 分，500-1500 字 1 分
- **权威栏目**（0-3 分）：社论/钟声 3 分，人民时评 2 分，观察/调查 1 分

> 评分是启发式打分，系数未经历史标定，仅供参考。优先度 ≥ 7 = 核心议题；4-6 = 观察议题；< 4 = 边缘议题。

重点关注 `step1_agenda.high_priority_articles`（≥7分），以及**连续多期跌出此列表**的议题（沉默信号）。

**Step 2 — 政策关联度检查**

目标议题是否有上位政策文件支撑：
- 当年政府工作报告关键词
- 国务院常务会议近期议题
- 中央经济工作会议定调
- 近期中央文件或领导人公开讲话

关联度高 = 政策方向不会轻易转向；关联度低 = 可能是临时议题或试探性信号。

---

### 第二层：版面·栏目层（看"怎么放"）

**Step 3 — 栏目温度判断**

| 栏目类型 | 信号含义 |
|---------|---------|
| 政论类（社论、评论员文章、钟声） | 官方立场确认，政策方向不可逆 |
| 民生调查类（调查、记者手记） | 问题导向，后续可能有监管动作 |
| 成就综述类（综述、系列报道） | 正面背书，政策红利期 |
| 批评曝光类（曝光、追问） | 风险信号，涉事行业/地区需警惕 |

**Step 4 — 地域映射分析**

从 `step4_regions.top_regions` 读取高频省市，结合报道语境判断：
- 某省/市频繁出现在负面报道 → 政策执行存在压力，可能成为整治样本
- 某省/市成为正面样本 → 具有政策先行价值，经验可能全国推广
- 某地区完全缺席 → 未被纳入议程（机会窗口或信息盲区）

### 第三层：语义·话语层（看"怎么说"）

**Step 5 — 语义三元组提取**

从目标文章提取高频叙事结构：
```
主语（谁）— 谓语（做了什么）— 宾语（对象/结果）
例：地方政府 — 切实落实 — 保交楼政策
例：部分企业 — 违规操作 — 监管红线
```
主语是"党/政府"时为施策主体叙事；主语是"群众/市场"时为问题呈现叙事。主语从政府切换到企业/市场，是叙事框架转变的早期信号。

**Step 6 — 话语强度七级定级（替代情感四象限）**

将目标议题的主要报道段落归入以下七级，输出各级占比和最高级别：

```
等级 1：研究探索   — "研究制定"、"鼓励试点"、"积极探索"
                     → 政策窗口开放，布局信号

等级 2：积极推进   — "加快推进"、"大力支持"、"全面部署"
                     → 政策红利期，正向信号

等级 3：规范引导   — "规范发展"、"健全机制"、"完善监管"
                     → 温和管控，行业自律期（12个月以上窗口）

等级 4：有序整治   — "专项整治"、"集中清理"、"有序规范"
                     → 中度管控，合规调整期（6-12个月窗口）

等级 5：坚决整治   — "坚决遏制"、"严格管控"、"决不允许"、"进一步规范"
                     → 强执行信号（3-6个月窗口）

等级 6：依法严惩   — "依法查处"、"追究责任"、"司法追诉"、"移送公安"
                     → 已进入法律轨道（1-3个月窗口）

等级 7：专项打击   — "雷霆行动"、"清网行动"、"扫黑除恶"、"集中收网"
                     → 运动式执法，无准备窗口
```

**输出格式（必须量化）：**
```
当期最高强度等级：N级（[触发词]）
各级段落占比：1级X% / 2级X% / 3级X% / 4级X% / 5级X% / 6级X% / 7级X%
较上期变化：最高等级 [上升N级 / 持平 / 下降N级]
跳级警报：[是否存在跳过中间等级的突变，如从2级直跳5级]
```

> **关键信号**：跳级突变（跨越2个以上等级）= 中央已决策，人民日报只是配合背书，时差极短（30天内）。连续两期等级上升且最高等级≥5 = 确认性整治信号。从6级回落到4级 = 整改期进入收尾，是政策松动的早期指标。
>
> 无历史数据时：必须标注"⚠️ 无基线，等级变化无法判断，仅输出当期快照"。

### ▶ 新增维度：部委协同度检测（横跨所有步骤）

**贯穿整个分析的核心维度，独立输出，不归属任何单一步骤。**

在所有文章（含 `official_media.json`）中，统计涉及的部委/机构类型：

| 协同等级 | 特征 | 信号含义 | 时间压缩 |
|---------|------|---------|---------|
| **L1 单部委** | 1个部委单独报道 | 行业关注，预警信号 | 无 |
| **L2 多部委** | 2-3个部委同时发声 | 跨部委协调期，行动概率显著提升 | -20% |
| **L3 国务院级** | 国务院常务会议议题 | 国家级执行意志 | -40% |
| **L4 政治局级** | 政治局/常委会出现 | 最高优先级，行动不可逆 | -60% |
| **L5 司法入轨** | 公安/检察/法院介入 | 已进入执法轨道，窗口关闭 | 窗口≤30天 |

从 `cross_validation.json` 的 `official_media` 或 WebSearch 结果中提取部委信号。若出现多部委联合发文或国务院专题部署，立即将风险等级上调一档，不受其他评分影响。

### ▶ 新增维度：政策时钟校正（用于风险窗口计算）

同样的话语强度信号，在不同年度节律阶段，实际行动时差不同：

| 月份区间 | 节律阶段 | 时钟系数 | 说明 |
|---------|---------|---------|------|
| 1-2月 | 人事调整期 | ×1.5（延长） | 大政策出台概率低，信号可靠性下降 |
| 3月 | 两会定调期 | ×1.0（基准） | 政府工作报告关键词是全年政策锚点 |
| 4-6月 | 政策部署期 | ×0.9 | 部委细化文件密集，信号是执行前奏 |
| 7-8月 | 北戴河敏感期 | ×1.3（延长） | 信号减少，不等于风险降低 |
| 9-10月 | 整治高发期 | ×0.5（压缩） | 专项行动高峰，与3月信号叠加大概率当年行动 |
| 11-12月 | 中经会定调期 | ×0.8 | 次年议题定调，前瞻价值最高 |

**时钟校正后窗口 = 基础窗口 × 时钟系数**

> 3月收到强度5级信号：基础窗口3-6个月 × 1.0 = 3-6个月  
> 9月收到相同信号：3-6个月 × 0.5 = 1-3个月（压缩至当年行动）

**Step 7 — 评论情感倾向（人工观察）**

在人民日报官方微博/微信评论区观察：
- 正面高赞评论 → 官方叙事获认同，舆情稳定
- 负面评论大量存在但未删除 → 官方允许有限异见，"引流泄压"机制
- 评论区关闭或删除率异常高 → 高敏感区域，存在重大舆情张力

**Step 8 — 舆论反弹度（估算）**

高反弹 = 负面评论 > 40% 且 24h 内删除率 > 30%（官方"想引导但引导失败"，政策落地阻力增加）

**Step 9 — 领导留言板线索（人工查询）**

访问 http://liuyan.people.com.cn，关注：
- 某类诉求大量提交但人民日报未跟进 → "官方知道但不愿放大"的信息管控区
- 留言板诉求被人民日报采写 → 该议题进入政策关注正轨

## 报告撰写原则

你是情报分析师，不是数据录入员。报告的价值是"差值"——官方决定让社会知道什么，与官方实际在想什么，之间的落差。

**动笔前问自己三个问题：**
1. 这期最出乎意料的信号是什么？
2. 如果我是这个行业的 CEO，最需要知道的那句话是什么？
3. 哪两个信号互相矛盾？矛盾本身说明了什么？

**文风四条：**
- 执行摘要写段落，不写项目符号
- 每个结论后接"这意味着……"
- 不确定时说"倾向于判断……，置信度有限"，不说绝对话
- 去掉"综合来看"、"如上所述"——每句都要有信息量

**触发问题：** "监管态度是否收紧？需要提前合规吗？"

## 重要局限（必须告知用户）

1. **信号效度未经验证**：框架打分规则是启发式的，未经历史案例回溯验证。在使用本框架做重大决策前，建议用已知结果的行业案例（教育双减、互联网反垄断）进行自我验证。

2. **基准率缺失**：任意行业在任意窗口内自然发生政策调整的概率未经统计。本框架的相对风险判断（本期 vs 上期）优于绝对风险判断。

3. **真实基层情绪被过滤**：人民日报反映官方视角，负面民意有结构性低估。必须交叉微博、知乎等平台。

4. **第四层数据缺失**：评论区情绪分析依赖人工观察，无自动化采集，可靠性受限。

5. **地区差异被平均化**：全国性报道掩盖地区执行差异，地方决策需补充本地媒体分析。

6. **政策时钟是统计规律，非铁律**：重大突发事件（经济危机、政治敏感周年）可能打乱正常节律。
